      Weak validation
      

      - excesive/invalid input value/s can cause the browser to stop working or shutdown.
      

      - application only uses weak limits/restrictions because of differences between user HW/SW capabilities, which have great effect on the behavior of the application. 
      

      
 
      Use console to monitor script (input) errors
      

      

      Mass download
      

      - mass download has to be allowed in the browser in order to use the "Download" button.    
      

      

      Known issues
      

      - mass and direct download doesn't work in some browsers.
